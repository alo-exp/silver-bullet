## Task 1 — Symlink hardening (Issue #25, SEC-02)

### Deliverable

New `hooks/lib/nofollow-guard.sh` exporting:

```bash
# shellcheck shell=bash
# Silver Bullet — refuse writes that would traverse a symlink.
# Sourced by every hook that creates, replaces, or appends to a state file
# under ~/.claude/.silver-bullet/.
sb_guard_nofollow() {
  local path="$1"
  if [[ -L "$path" ]]; then
    printf 'ERROR: refusing to write through symlink: %s\n' "$path" >&2
    exit 1
  fi
}

# Convenience: unlink a symlink squatting at path, then caller writes normally.
sb_safe_write() {
  local path="$1"
  [[ -L "$path" ]] && rm -f -- "$path"
}
```
