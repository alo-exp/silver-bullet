<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| bash → installed_plugins.json | jq writes to user's plugin registry; malformed JSON or mid-write crash could corrupt it |
| bash → plugin cache filesystem | rm -rf on a stale cache path; must be path-guarded to prevent accidental deletion |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-053-01 | Tampering | installed_plugins.json stale-key removal | mitigate | Atomic tmpfile+mv pattern in Step 6a prevents partial writes; file existence checked with jq -e before write |
| T-053-02 | Tampering | rm -rf stale cache in Step 6b | mitigate | Path guarded: only removes if path starts with $HOME/.claude/plugins/cache/silver-bullet/silver-bullet; -d check before rm -rf |
| T-053-03 | Elevation of Privilege | marketplace install command | accept | claude mcp install is the official CLI; no additional privilege beyond what Claude CLI already has; user confirmed install in Step 4 |
| T-053-04 | Information Disclosure | Error output from failed marketplace install | accept | Error shown to user is from claude CLI itself, not user-controlled input; low risk |
</threat_model>
