**File:** `~/.claude/hooks/gsd-read-guard.js`

Replace the current advisory logic with PPID-based session deduplication:

1. Remove the `CLAUDE_SESSION_ID` env var bypass (lines 39-42) — it's unreliable
2. Before the advisory output section (currently lines 63-77), add:

```javascript
// HOOK-05: Deduplicate advisory per file per session (PPID = session key)
const sessionDir = `/tmp/gsd-rg-${process.ppid}`;
const safeFileName = Buffer.from(filePath).toString('base64').replace(/[/+=]/g, '_').slice(0, 200);
const trackFile = path.join(sessionDir, safeFileName);
try { fs.mkdirSync(sessionDir, { recursive: true, mode: 0o700 }); } catch {}
if (fs.existsSync(trackFile)) {
  process.exit(0);
}
try { fs.writeFileSync(trackFile, '1', { flag: 'wx' }); } catch {}
```
