## Task 1 — Fix P1 bugs in spec-floor-check.sh (items 1, 2)

**Files modified:**

### 1a. Replace `\b` word boundaries with POSIX-compatible patterns (lines 44, 46)

`\b` is PCRE-only and fails silently with macOS stock `grep -E`. Replace with `grep -wE`
which uses word boundary matching in POSIX grep.

**Line 44** — change:
```bash
if printf '%s' "$cmd" | grep -qE '\bgsd-plan-phase\b|\bgsd[- ]plan[- ]phase\b'; then
```
to:
```bash
if printf '%s' "$cmd" | grep -qwE 'gsd-plan-phase|gsd[- ]plan[- ]phase'; then
```

**Line 46** — change:
```bash
elif printf '%s' "$cmd" | grep -qE '\bgsd-fast\b|\bgsd[- ]fast\b|\bgsd[- ]quick\b'; then
```
to:
```bash
elif printf '%s' "$cmd" | grep -qwE 'gsd-fast|gsd[- ]fast|gsd[- ]quick'; then
```

### 1b. Fix `^## Overview` prefix match (line 63)

Change `"^${section}"` to `"^${section}$"` so `## Overview` does not match `## Overviewer`.

**Line 63** — change:
```bash
if ! grep -q "^${section}" "$SPEC"; then
```
to:
```bash
if ! grep -q "^${section}$" "$SPEC"; then
```
