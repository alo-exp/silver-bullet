## What Was Built

`/silver` is a pure dispatcher skill. It:

- Shows a routing banner (input, destination, reason) before every invocation
- Asks a clarifying AskUserQuestion when input is ambiguous (top 2-3 options)
- Routing priority: SB skills first, GSD fallback, clarification last
