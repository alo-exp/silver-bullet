# Silver Skill Scenario

## Skill: silver
## Context: Intent-based workflow routing

### Scenario: Route user intent to appropriate workflow

**Trigger Phrases:**
- "I need to"
- "help me"
- "build"
- "create"
- "add"

**Silver Routes:**
1. User says "I need to add a new API endpoint" → silver-feature
2. User says "the login is broken" → silver-bugfix
3. User says "improve the UI" → silver-ui
4. User says "setup CI" → silver-devops
5. User says "research databases" → silver-research
